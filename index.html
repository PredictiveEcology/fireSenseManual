Overview

This manual is a live document which is automatically updated as changes are made to underlying model code and documentation.

The fireSense model implements data-driven wildfire ignition, escape, and spread sub-models which are sensitive to climate and vegetation feedbacks (Marchal et al. 2017b, a; 2019). There are two stages to the modelling workflow: fitting the corresponding statistical models for each process, and subsequently using these models for prediction.
